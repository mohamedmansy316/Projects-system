## Projects system
### Prerequisites

- PHP 7.4
- Node 16.14.2
- Composer 2
### Installation

   - composer install
   - npm install
   - add database with name (project-management)
   -  php artisan passport:client --personal

Import the file with name "Projects management" into postman
